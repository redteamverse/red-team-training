Course Curriculum

What You'll Cover During Training

The CRTP training is organized into four modules across three days, covering enumeration, privilege escalation, persistence, and enterprise attack paths in Active Directory environments.

  • Active Directory enumeration of users, groups, trusts and ACLs
  • Mapping attack paths across enterprise environments
  • Local privilege escalation techniques on Windows systems
  • Discovering machines where domain users have local admin access
  • PowerShell based reconnaissance techniques
  • Modifying tools to bypass Windows Defender
  • Bypassing AMSI, Script Block Logging and CLM
  • Using .NET loaders to execute assemblies in memory
  • Credential harvesting and session discovery
  • Credential replay attacks for privilege escalation
  • Extracting credentials in restricted environments
  • Kerberoasting attacks and variants
  • Exploiting delegation misconfigurations
  • Privilege escalation through protected groups
  • Persistence using Golden, Silver and Diamond tickets
  • Skeleton key and custom SSP persistence techniques
  • AdminSDHolder persistence abuse
  • DCSync attacks through ACL abuse
  • Modifying domain controller security descriptors
  • Escalating privileges from Domain Admin to Enterprise Admin
  • Intra-forest trust exploitation techniques
  • Database link abuse for cross-forest execution
  • Active Directory Certificate Services attack paths
  • Cross-domain trust attacks to escalate privileges
  • Events generated during common Active Directory attacks
  • Security architecture improvements and defensive controls
  • LAPS, PAW, Tiered Administration and ESAE models
  • Credential Guard and WDAC security mechanisms
  • Understanding Microsoft Defender for Endpoint telemetry
  • Bypassing MDE detection mechanisms
  • Microsoft Defender for Identity detection logic
  • Evasion techniques against MDI anomaly detection
