Request Approval to Attend AltSecCON

AltSecCon is an in-person red teaming training event focused on enterprise attack techniques in Active Directory and Azure environments. If you need approval from your manager, the points below explain the value of attending.

Strengthen Enterprise Security

Learn how attackers compromise enterprise Active Directory and Azure environments and understand how to identify and mitigate these weaknesses.

Hands-on Red Team Training

Practice real-world offensive techniques through guided labs designed to simulate enterprise attack scenarios.

Identify Security Gaps

Understand common misconfigurations and privilege escalation paths attackers use in corporate environments.

Bring Knowledge Back

Apply the techniques learned during the training to improve your organization's security posture and detection capabilities.

Key benefits include:

Event Format

Event Structure Overview

AltSecCON was conducted as a focused, in-person Red Team training event structured for depth, continuity, and practical execution.

Multi-Day Format

Training was delivered across multiple consecutive days to allow uninterrupted learning.

Full-Length Courses

Each program was taught end-to-end rather than split into short sessions.

Parallel Training Tracks

Multiple courses ran simultaneously to support different experience levels.

In-Person Delivery

All sessions were conducted on-site with direct instructor interaction.

Daily training hours: 9:00 AM – 5:00 PM
Scheduled breaks: Tea breaks and lunch included
