Who Should Attend

The AltSecCON Practitioner Conference is designed for security professionals, researchers, and practitioners interested in offensive security, enterprise defense strategies, and real-world cybersecurity challenges.

Red Team Operators

Professionals involved in adversary simulation, security assessments, and Red Team engagements within enterprise environments.

Penetration Testers

Security professionals performing vulnerability assessments and penetration testing across applications, networks, and cloud environments.

Security Engineers

Engineers responsible for protecting enterprise infrastructure, monitoring threats, and improving defensive capabilities.

Security Researchers

Security analysts working in Security Operations Centers monitoring attacks, investigating threats, and improving detection capabilities.
