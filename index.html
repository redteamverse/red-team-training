Course Content

Explore our comprehensive curriculum covering all aspects of Active Directory and cloud security for red team operations.

  • Extensive Active Directory enumeration

    Learn advanced techniques to map out Active Directory environments, identify security weaknesses, and discover potential attack paths using both built-in and custom tools.

  • Active Directory trust mapping and abuse

    Discover how to identify, map, and exploit trust relationships between domains and forests to expand access and move laterally through complex environments.

View full module details

  • Privilege escalation via user hunting and misconfigurations

    Master techniques to identify and exploit common privilege escalation paths through user hunting, group membership analysis, and security misconfigurations in Active Directory.

  • Abusing delegation issues (Unconstrained, Constrained, RBCD)

    Learn how to identify and exploit various delegation models including unconstrained delegation, constrained delegation, and resource-based constrained delegation to elevate privileges.

View full module details

  • Kerberos attacks: Golden ticket, Silver ticket, Kerberoasting

    Understand and execute advanced Kerberos-based attacks including golden tickets, silver tickets, kerberoasting, and AS-REP roasting to compromise authentication systems.

  • Credential replay: Over-PTH, token replay, and more

    Master various credential replay techniques including Pass-the-Hash, Pass-the-Ticket, token impersonation, and other methods to reuse compromised credentials for lateral movement.

View full module details

  • Lateral movement and privilege escalation across forest trusts

    Learn techniques to identify and exploit trust relationships between forests, enabling lateral movement and privilege escalation across organizational boundaries.

  • Exploiting trusts in Microsoft Exchange, SQL Server, etc.

    Discover how to leverage trust relationships in enterprise applications like Microsoft Exchange and SQL Server to gain unauthorized access and escalate privileges.

View full module details

  • DCShadow, WMI, GPO, Domain/Host ACLs and more

    Master advanced persistence techniques including DCShadow for stealthy directory modifications, WMI event subscriptions, Group Policy Object manipulation, and access control list modifications.

View full module details

  • Monitoring AD activities and deception tactics

    Learn how to monitor Active Directory for suspicious activities and implement deception tactics to detect and mislead attackers while maintaining operational security.

  • Defense bypassing: Red Forest, JEA, LAPS, ATA, App Whitelisting, and more

    Master techniques to bypass modern defensive controls including Red Forest architecture, Just Enough Administration, Local Administrator Password Solution, Advanced Threat Analytics, and application whitelisting.

View full module details
